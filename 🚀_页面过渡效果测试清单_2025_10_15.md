### ✅ 必须通过

### ⚠️ 可选优化
1. 如果觉得过渡太慢：改为 `0.2` 秒
2. 如果觉得过渡太快：改为 `0.4` 或 `0.5` 秒
3. 想要更华丽效果：添加轻微滑动（见文档）

## 🔧 如果发现问题

### 过渡效果不明显
```typescript
// 增加过渡时长
const pageTransition = {
    duration: 0.5,  // 改为 0.5 秒
    ease: "easeInOut"
};
```

### 过渡太慢
```typescript
// 减少过渡时长
const pageTransition = {
    duration: 0.2,  // 改为 0.2 秒
    ease: "easeInOut"
};
```

### 想要滑动效果
```typescript
// 添加垂直滑动
const pageVariants = {
    initial: { opacity: 0, y: 10 },
    animate: { opacity: 1, y: 0 },
    exit: { opacity: 0, y: -10 }
};
```
