## 📐 图片规格

- **分辨率**: 最低 1920x1080 (建议 2048x1536)
- **格式**: PNG (首选) 或 JPEG
- **纵横比**: 16:9 或 4:3
- **质量**: 专业摄影级别
- **风格**: 必须与提示词描述完全匹配

## 📤 上传和更新流程

### 步骤1: 生成图片
使用每个模板的提示词生成对应图片

### 步骤2: 上传到存储
选择一个存储方案：
- Supabase Storage (推荐)
- Google Cloud Storage
- AWS S3
- 其他CDN

### 步骤3: 更新数据库

#### 方法A: 使用 SQL (批量更新)
```sql
-- 示例：更新 Exterior Design
UPDATE design_templates 
SET image_url = 'https://your-storage.com/modern-contemporary-exterior.jpg'
WHERE name = 'Modern Contemporary Exterior' 
  AND main_category = 'Exterior Design';

-- 重复以上操作为每个模板更新图片URL
```

#### 方法B: 使用 Admin Panel
如果您的 Admin Panel 有编辑模板功能，可以：
1. 登录 Admin Panel
2. 找到对应模板
3. 编辑并上传图片
4. 保存
